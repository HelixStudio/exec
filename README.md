# exec
Yet another *blazingly fast* code execution engine written in Rust.

## Support
 - C
 - C++
 - Rust
 - Haskell

## Paths
 - GET /api/v1/status
 - GET /api/v1/runtimes
 - POST /api/v1/execute
 - POST /api/v1/test

Check the `models.rs` file for request & response models.
